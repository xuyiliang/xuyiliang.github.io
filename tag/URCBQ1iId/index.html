每日一题 leetcode11 装最多水的容器

2020-04-18 #算法 #数据结构
直接放题目，这道题应该就比较熟悉了 给你 n 个非负整数 a1，a2，...，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直 线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。 说明：你不能倾斜容器，且 n 的值至少为 2。 首先先明白题意，就是给一个数组，数组中的数字代表着柱子的高度，然后需要找到两根柱子，他们之间能装的水最多，返回水量。 那么水量=两根柱子较短的那根*两根柱子之间的距离 明白这个之后，就可以做题了 这道题的话采用双指针+贪心，设定left=0,right=height.length-1,然后用一个数来存放结果，那么柱子left和right如何变化呢？ 如果height[left]&lt;height[right],那么left++，反之则right++，因为水能够存放多少是由短的那个柱子决定的，就是所谓的木桶短板效应了。 直接贴代码 class Solution { public static int maxArea(int[] height) { if(height.length==0 || height.length==1){ return 0; } int max = 0; int left = 0; int right = height.length-1; while(left&lt;right){ int cur = Math.min(height[left],height[right])*(right-left); max = cur&gt;max?cur:max; if(height[left]&lt;height[right]){ left++; }else{ right--; } } return max; } }

每日一题 leetcode69.x的平方根

2020-04-18 #算法 #数据结构
今天开始更新做数据结构算法题的感想与收获吧！ 题目： 实现 int sqrt(int x) 函数。 计算并返回 x 的平方根，其中 x 是非负整数。 由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。 这道题就是让我们自己实现Math.sqrt()函数，一拿到，有没有想直接调个函数，返回，但是不可取哦，真的面试可不能这样。 说一下拿到这道题的思路，一开始我是想着，输入一个数x，那就用for循环从i=0到i=x/2+1进行判断，然后如果***i的平方&lt;=x &amp;&amp; (i+1)的平方&gt;=x***那就说明找到了；但是这个想法有两个问题，第一个是时间太久了，第二个是，i的平方有可能int装不下，那么此时想到把判断***i的平方&lt;=x***转化成***i&lt;=x/i ***这样就不用去管会不会超出范围了。 那时间太久呢？采用二分法，由于x是Int，所以最大2147483647开平方46340.95，所以只要在0~46341间做判断就行了。代码如下 class Solution { public static int mySqrt(int x) { int res = 0; if(x==0) return res; int left = 0; int right = 46341; while(left&lt;right){ int mid = left + (right - left)/2; if(x/mid&gt;=mid &amp;&amp; x/(mid+1)&lt;=(mid+1)){ if(x/(mid+1)==(mid+1)){ return mid+1; }else{ return mid; } }else if((mid+1)&lt;x/(mid+1)){ left = mid+1; }else{ right = mid; } } return res; } }
