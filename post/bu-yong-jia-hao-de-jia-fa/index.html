每日一题 leetcode 17.01.不用加号的加法

2020-04-22

设计一个函数把两个数字相加。不得使用 + 或者其他算术运算符。

示例:
输入: a = 1, b = 1
输出: 2

这道题是简单题，不过得了解一些异或操作才能做出来。
思路就是用^存放值，&和<<进位

class Solution {
    public int add(int a, int b) {
        int sum = 0;
        int carry = 0;
        while(b!=0){
            sum = a ^ b;
            carry = (a & b)<<1;
            a = sum;
            b = carry;
        }
        return a;
    }
}
